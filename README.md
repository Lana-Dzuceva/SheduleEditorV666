# Разработка редактора расписания для факультета МиКН

Крутой редактор расписания, написанный в соло by Лана Дзуцева. В приложении реализованы:
- разбиение на подгруппы и недели,
- сохранение после любого изменения
- автоматическая проверка на множество различных ошибок
- подсветка ошибок
- список ошибок с возможностью быстрого перехода между ними
